## How I work

![Outcome over Output](https://img.shields.io/badge/Mindset-Outcome%20over%20Output-2E7D32?style=flat-square)
![Feedback Loops](https://img.shields.io/badge/Practice-Short%20Feedback%20Loops-2E7D32?style=flat-square)
